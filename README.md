# Preston Jay Susanto - Elite Portfolio
https://shadowesu.github.io/Preston_Susanto_Portfolio/

A high-performance personal portfolio website built with pure Vanilla JS, CSS3, and HTML5.

## Project Structure
- `index.html`: Main semantic structure and resource linking.
- `style.css`: Modern minimalist styling with custom variable systems and transitions.
- `script.js`: Data-driven rendering engine and interactive UI logic.
- `README.md`: Project documentation.

## Key Features
- **Global Journey Timeline**: Tracking growth across four nations (Indonesia, China, Australia, USA).
- **Verifiable Proof Mode**: Highlighting academic excellence (10 A's) and technical credentials.
- **Licenses & Certifications**: Includes Google x NexusEdge foundations in Generative AI, LLMs, and Python.
- **Accessibility Focus**: Built with WCAG principles in mind, inspired by "Access for All" hackathon wins.
- **Responsive Design**: Fluid layouts optimized for all devices.

## Deployment
This project is ready for **GitHub Pages**, **Netlify**, or **Vercel**. Simply upload these four files to the root of your repository.
